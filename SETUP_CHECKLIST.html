🤖BookStore Chatbot Setup

Gemini AI + Database Integration

📋Phase 1: Prerequisites

💡 Tip: Save your Gemini API Key in a safe place. You'll need it in the next step.

⚙️Phase 2: Backend Setup

cd backend
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/Mac:
source venv/bin/activate
pip install -r requirements.txt
# backend/.env
GEMINI_API_KEY=your_actual_api_key_here
python manage.py migrate
python manage.py runserver
# Server at http://localhost:8000
✓ Backend is running! Server should be at http://localhost:8000/api/

🎨Phase 3: Frontend Setup

cd frontend
npm install
# .env
VITE_API_URL=http://localhost:8000/api
npm run dev
# Frontend at http://localhost:5173
✓ Frontend is running! Open http://localhost:5173 in your browser.

🧪Phase 4: Testing

🧪 Test Messages:
- "Bạn có sách gì?" (List books)
- "Sách của tác giả nào?" (By author)
- "Sách giá rẻ nhất" (Cheapest)
- "Cách đặt hàng?" (How to order)

✅Phase 5: Verification

Environment Variables Checklist:
  • ✓ GEMINI_API_KEY set in backend/.env
  • ✓ VITE_API_URL set in frontend/.env
  • ✓ Backend server running on :8000
  • ✓ Frontend dev server running on :5173
  • ✓ No CORS errors in browser console

🆘Troubleshooting

❌ "GEMINI_API_KEY not found"
Solution: Check .env file has GEMINI_API_KEY set
❌ "Connection refused localhost:8000"
Solution: Ensure Django server is running: python manage.py runserver
❌ "CORS error from frontend"
Solution: Backend already configured, check VITE_API_URL
❌ "Chat not responding"
Solution: Check API key validity, check browser console for errors
